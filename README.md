# OnlyFacts — מנפצי השקרים

אתר קטן להצגת הוכחות חותכות וטיפול בטענות שגויות. הקבצים הראשיים נמצאים בשורש הפרויקט:

- `index.html` — דף הבית והמערכת לפרסום פוסטים
- `styles.css` — עיצובים
- `app.js` — לוגיקת צד לקוח, שמירת פוסטים ב-`localStorage`

הוראות הרצה מקומית

1. פשוט פתחו את הקובץ `index.html` בדפדפן:

```bash
# בדפדפן: פתחו את index.html ישירות
```

2. או הריצו שרת סטטי פשוט (מומלץ לשים לב למדיניות CORS אם מוסיפים מקורות חיצוניים):

```bash
# עם Python 3
python3 -m http.server 8000
# ואז פתחו http://localhost:8000
```

מאפיינים

- שמירה מקומית של פוסטים ב-`localStorage` תחת המפתח `onlyfacts_posts_v1`.
- חיפוש לפי טקסט
- מחיקת פוסט או מחיקה כוללת

העלאה לפרודקשן

- ניתן לפרסם את האתר ב-GitHub Pages או בכל שרת סטטי. פשוט דחפו את התיקייה ל-repo ופעילו Pages.

רעיונות להמשך

- הוספת אימות/התחברות לנהל פוסטים
- ממשק עריכה
- אפשרות לייצא/לייבא פוסטים כ-JSON

מחקרים ודחיפה לריפוזיטורי

- הוספתי מדור "מאגר מחקרים" עם 18 רשומות של מקורות ציבוריים ב-`data/studies.json`.
- כדי לדחוף את כל התיקייה ל-GitHub (אתה נתת הרשאה), הפקודות שהיית צריך להריץ במכונה שלך:

```bash
git add .
git commit -m "Add research library, UI and styles"
git push origin main
```

אם תרצה, אני יכול להכין גם workflow ל-GitHub Actions לפריסה אוטומטית ל-GitHub Pages (אצטרך אישור להוסיף תיקיית `.github/workflows`).
